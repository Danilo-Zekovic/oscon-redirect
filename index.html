SPA Bootcamp Instructions

Hello! There will be two ways to run our SPA and access data; first fetches data from online server and second one accesses local database. To be able to run the code and follow up with the tutorial you have to set up the development enviroment.

In order to have the best experience at our SPA Bootcamp, we ask that you attempt to download and install the following programs before our presentation on May 17, 2016.

Requirements

  • Node
  • MongoDB (Optional if you would like to keep image data local)

Vagrant

Follow these steps to get your Vagrant machine running

  1. Download and unzip our custom Vagrant box >>>>DOWNLOAD LINK (~120MB) <<<<
  2. Bring up the box (vagrant up); allow installation to complete
  3. Halt the box (vagrant halt)
  4. Install the guest additions as follows:
    In the VirtualBox console, select the oscon-box -> settings -> storage -> add new storage attachment -> add optical drive -> leave it empty
  5. Reboot (vagrant up)
  6. Login screen, leave everything as it is and use password "vagrant"
  7. Top menu -> Devices -> Insert Guest additions CD image
  8. sudo mkdir /mnt/cdrom
  9. sudo mount /dev/sr0 /mnt/cdrom
  10. cd /mnt/cdrom
  11. sudo ./VBoxLinuxAdditions.run (NOTE: "dot-slash" at beginning)
  12. sudo umount /mnt/cdrom
  13. On your host machine to shut down run :$ vagrant halt
