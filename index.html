SPA Bootcamp Instructions

Hello! There will be two ways to run our SPA and access data; a downloaded local version or a pre-constructed cloud version. The local version will require a few building steps, while the cloud is pre-constructed and will not require any extra work.

In order to have the best experience at our SPA Bootcamp, we ask that you attempt to download the following programs before our presentation on May 15, 2016. These programs are:

Requierments

  • Node
  • MongoDB (Only if you want to use local assets, data)

Vagrant

  1. Download and unzip our Vagrant box >>>>DOWNLOAD<<<<
  2. You have to boot twice (vagrant up; vagrant halt)
  3. VirtualBox Console, select the oscon-box -> settings -> storage -> add new storage attachment -> add optical drive -> leave it empty
  4. vagrant up
  5. Login screen, leave everything as it is and use password "vagrant"
  6. Top menu -> Devices -> Insert Guest additions CD image
  7. sudo mkdir /mnt/cdrom
  8. sudo mount /dev/sr0 /mnt/cdrom
  9. cd /mnt/cdrom
  10. sudo ./VBoxLinuxAdditions.run (not dot-slash at beginning)
  11. sudo umount /mnt/cdrom
  12. On your host machine to shut down run :$ vagrant halt
